Программа скачивает видео с Ютуба по ссылке. 

Плюсы:
+ мало весит
+ быстро работает
+ в папке dist лежит исполняемый exe файл, который работает отдельно -> бери, клади куда хочешь
+ не надо устанавливать

Минусы:
- баг при неправильно введении адреса видео - может вылетать
- при отсутствии интернета тоже вылетит
